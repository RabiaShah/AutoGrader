Requirements:
-MongoDB Database
-Python 3.6+
-Flask Web Framework
-rest of the code requirements are listed in detail in the requirements.txt file


Instructions:
1. Install MongoDB and import all the files in the "Mongo Database" folder.
2. Run mongod.exe and then mongo.exe from the bin folder of the MongoDB.
3. Open "autograder.py" in a code editor to view the code (this step is optional)
4. Run the command in the cmd "python autograder.py"
5. When the command is running in the localhost, go to your browser and type "localhost:5000" in the URL, or whatever port
   at which the project is running on.
6. The usernames and passwords of instructors and different students are given in "Info" database. Use them to login.
